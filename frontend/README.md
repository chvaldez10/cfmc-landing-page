## Frontend 🖥️

Frontend using Next.js and React with App Router.

## Folder Structure

```
/app
│
├── src
│   ├── page.tsx        # Maps to the / route
│   ├── layout.tsx
│   ├── globals.css
│   └── about           # Maps to the /about route
│       └── page.tsx
├── components
│   ├── Header.tsx
│   └── Footer.tsx
├── public              # Static files
│   └── assets
│       ├── images
│       └── fonts
├── styles
│   └── *.css
├── lib
│   └── *.tsx
├── hooks
│   └── *.tsx
├── utils
│   └── *auth*.tsx
└── next.config.tsx
```
